1.1. El siglo de los datos

Actualmente hay harto dato pululando por ahí. Esto se debe a:

  1. Hay mayor cantidad de datos que se producen y procesan. Aparte, estos datos son más complejos y menos estructurados. Los datos cumplen las 3V: 1. Volumen: Los datos tienen tamaños de 10^12 bytes a 10^18 bytes 2. Velocidad: Los datos se producen en gran velocidad 3. Variedad: Los datos son de diversos tipos, y pueden tener estructura o no. Para poder procesarlos, se necesitan sistemas automático que simplifiquen el proceso

  2. Mejora en el hardware que permiten el procesamiento de datos. Uso de GPU's

  3. Cloud computing, reduce los costos de las weas y hace todo mas barato y sencillo de iniciar

  4. Facilidad de acceso al desarrollo de software: compus más baratas, más tutoriales y más cosas para aprender. Se recomienda otros lugares para aprender: Repositorio de muchas cosas y Repositorio de Python y así

Los datos son la materia prima para la AI. Se origina el término en 1956, por parte de John McCarthyy es la capacidad de las computadoras de pensar como los humanos para resolver problemas . La AI utiliza datos de entrada y salida, esta aproximación se conoce como Machine learning y es como las máquinas aprenden.
