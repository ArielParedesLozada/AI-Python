1.1. El siglo de los datos

Actualmente hay harto dato pululando por ahí. Esto se debe a:

  1. Hay más datos menos organizados que procesar. Cumplen las siguientes características:
    • Volumen: Datos grandes
    • Velocidad: Se producen rápido
    • Variedad: Datos diversos
  2. Mejora en el hardware que permiten el procesamiento de datos. Uso de GPU's para más potencia en el procesamiento.
  3. Cloud computing, reduce los costos de las weas y hace todo mas barato y sencillo de iniciar. Se piden prestados servicios e infraestructura a empresas más grandes para tener un inicio
  4. Facilidad de acceso al desarrollo de software: compus más baratas, más tutoriales y más cosas para aprender. Se recomienda otros lugares para aprender: Repositorio de muchas cosas y Repositorio de Python y así

Los datos son la materia prima para la AI. Es la capacidad de sistemas de pensar ocmo humanos para llegar a soluciones.

Machine Learning

Es el modo por el cual las máquinas aprenden. Consiste en un conjunto de datos de entrada y salida que se usan para entrenar a la AI.
Los datos de entrada le enseñan a la AI que hay que hacer, y la AI produce datos de salida, las respuestas.
La AI moderna tiene millones de datos de entrada, con millones de datos de salida y miles de parámetros. La wea pulenta.
